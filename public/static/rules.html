HackTalk rules

There are multiple rules you must follow in order to use HackTalk safely and help other users:

  1. You must not spam; doing so will result in a ban.
  2. Be thoughtful with your answers: don't give a short, useless answer.
  3. Give as much information as you can with your questions. Attaching a link to a Repl or including code snippets is recommended.
  4. No profanity. If you censor profanity (eg. "f**k") or not type it explicitly (eg. "WTF"), it is permitted, but discouraged.
  5. No bullying will be tolerated and doing so will result in an immediate ban.
